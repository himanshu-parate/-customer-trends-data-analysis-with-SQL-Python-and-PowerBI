# 👨🏻‍💻 Customer Trends Data Analysis | SQL • Python • Power BI

This project showcases a complete **end-to-end Data Analytics workflow** built using **Python, SQL, and Power BI** to analyze customer shopping behavior, purchasing trends, and business performance insights.

The project simulates a real-world analytics environment where raw customer transaction data is transformed into actionable business intelligence through data cleaning, SQL analysis, dashboarding, and reporting.

This project is ideal for:

* 📊 Aspiring Data Analysts building strong portfolio projects
* 🐍 Learners practicing Python for data cleaning & EDA
* 🗄️ SQL learners solving real business problems
* 📈 Power BI users creating interactive dashboards
* 💼 Students & professionals preparing for Data Analytics interviews

---

# 📌 Project Overview

The main objective of this project is to analyze customer behavior and purchasing patterns to help businesses make data-driven decisions.

The project covers:

✅ Data Cleaning & Exploratory Data Analysis using **Python**

✅ Business Problem Solving using **SQL Queries**

✅ Interactive Dashboard Creation using **Power BI**

✅ Data Visualization & KPI Reporting

✅ Business Insights & Recommendations

# 🗄️ SQL Analysis

The SQL section focuses on solving business problems using analytical SQL queries such as:

```sql

FROM customer_data
GROUP BY gender;
```

# 🚀 How to Run This Project

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/customer-trends-data-analysis-SQL-Python-PowerBI.git

cd customer-trends-data-analysis-SQL-Python-PowerBI
```

---

## 2️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Run Jupyter Notebook

Open:

```bash
customer_trends_analysis.ipynb
```

This notebook contains:

* Data Cleaning
* EDA
* Visualization
* SQL Export Process

---

## 4️⃣ Run SQL Queries

* Create a database in MySQL / PostgreSQL / SQL Server
* Import cleaned dataset
* Execute queries from:

```bash
customer_trends_queries.sql
```

---

## 5️⃣ Open Power BI Dashboard

Open:

```bash
customer_trends_dashboard.pbix
```

Explore interactive business insights and dashboards.

# 📌 Key Insights

* Customers aged between **25–40** contribute the highest revenue.
* Discounts significantly increase purchase frequency.
* Female customers showed higher average spending in selected categories.
* Electronics and Fashion categories generated maximum sales.
* Digital payment methods are preferred over cash transactions.
